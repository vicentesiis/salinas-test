* Arquitectura: Utilice MVC, ya que es la arquitectura que te provee Apple nativamente y la más fácil de implementar para este proyecto.

* Librerías:
	- Alamofire: Se encarga de realizar las peticiones HTTP a el servidor, agiliza la tarea de el consumo de Rest APIs.
	- Realm: Librería para persistir los datos, es mucho más rápido que SQLite y Core Data.

	- Aprovechamiento de el controlador de versiones.

* Problemas encontrados:
